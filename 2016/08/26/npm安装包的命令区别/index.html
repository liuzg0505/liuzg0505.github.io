npm安装包的命令区别

npm install 包三种方式的区别

#npm install

本地安装，安装到当前项目的 node_modules/ 目录下面
包依赖 不会 加到 package.json 文件的 dependencies 中，也不会 加到 devDependencies 中。

#npm install –save

简写： npm install -S
本地安装，安装到当前项目的 node_modules/ 目录下面，
包依赖 会 加到 package.json 文件的 dependencies 中，但 不会 加到 devDependencies 中。

dependencies是运行时依赖：

devDependencies是开发时的依赖：

#npm install –save-dev

简写：npm install -D
包依赖 会 加到 package.json 文件的 dependencies 中，也 会 加到 devDependencies 中。
